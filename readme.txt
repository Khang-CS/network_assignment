Để chạy được chương trình, Thiết bị cần được cài môi trường JAVA
File chạy được đặt trong thư mục jarFile:
- server.jar: dùng để chạy và khởi động server.
- client.jar: dùng để chạy và khởi động client.

Tên của port phải là số nguyên, port mặc định là 3200.


Source Code được đặt trong 2 thư mục client/java và server000/java
client/java là source code của phần client
server000/java là source code của phần server.
